# react-native-simple-mopub

MoPub SDK for RN

## Installation

```sh
npm install react-native-simple-mopub
```

## Usage

```js
import SimpleMopub from "react-native-simple-mopub";

// ...

const result = await SimpleMopub.multiply(3, 7);
```
